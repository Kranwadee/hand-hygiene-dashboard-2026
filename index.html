ระบบติดตามความสะอาดมือ
Hand Hygiene Monitoring System

ระบบบันทึกและวิเคราะห์ความสะอาดมือสำหรับบุคลากรทางการแพทย์
ช่วยลดการแพร่กระจายเชื้อโรคในโรงพยาบาล

งานป้องกันและควบคุมการติดเชื้อ (Infection Control)

IC Dashboard

ภาพรวมโรงพยาบาล (Hospital Overview)

สำหรับ: ผู้ดูแลระบบ / IC Team

ดูภาพรวมความสะอาดมือทั้งโรงพยาบาล วิเคราะห์ข้อมูลทุกแผนก ทุกตำแหน่ง พร้อมรายงานสรุป

Department Dashboard

ข้อมูลแผนก (Department View)

สำหรับ: หัวหน้าแผนก / บุคลากรในแผนก

ติดตามผลงานบุคลากรในแผนก ดูแนวโน้มประสิทธิภาพ และจัดอันดับคะแนนความสะอาดมือ

Personal Dashboard

ข้อมูลส่วนบุคคล (Personal View)

สำหรับ: แพทย์ / พยาบาล / บุคลากร

ดูผลการสแกนมือส่วนตัว และวิเคราะห์โซนที่ต้องปรับปรุง
